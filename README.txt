DESCRIPTION

A simple gallery based on a Flickr group where images are posted by individual contributors and by the group owner on behalf of other contributors.

GET IT

From GitHub: 
http://github.com/MagnusEnger/flickrgroupgallery/tree/master

Or, using git:
git clone git://github.com/MagnusEnger/flickrgroupgallery.git

INSTALLATION

- Dump all files in a directory. 
- Rename config-dist.php to config.php
- Edit config.php

EXAMPLE

http://lundehund.no/galleriet/
